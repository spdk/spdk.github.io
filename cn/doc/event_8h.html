◆ spdk_app_start()

int spdk_app_start ( struct spdk_app_opts *  opts,
spdk_event_fn  start_fn,
void *  arg1,
void *  arg2 
)

Start the framework.

Before calling this function, the fields of opts must be initialized by spdk_app_opts_init(). Once started, the framework will call start_fn on the master core with the arguments provided. This call will block until spdk_app_stop() is called, or if an error condition occurs during the intialization code within spdk_app_start(), itself, before invoking the caller's supplied function.

Parameters
optsInitialization options used for this application.
start_fnEvent function that is called when the framework starts.
arg1Argument passed to function start_fn.
arg2Argument passed to function start_fn.
Returns
0 on success or non-zero on failure.

◆ spdk_app_start_shutdown()

void spdk_app_start_shutdown ( void  )

Start shutting down the framework.

Typically this function is not called directly, and the shutdown process is started implicitly by a process signal. But in applications that are using SPDK for a subset of its process threads, this function can be called in lieu of a signal.

◆ spdk_app_stop()

void spdk_app_stop ( int  rc)

Stop the framework.

This does not wait for all threads to exit. Instead, it kicks off the shutdown process and returns. Once the shutdown process is complete, spdk_app_start() will return.

Parameters
rcThe rc value specified here will be returned to caller of spdk_app_start().
