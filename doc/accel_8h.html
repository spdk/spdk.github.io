Append a compare operation to a sequence.

This operation compares data from two source buffers to verify they are identical. If a mismatch is found, the operation will typically complete with a failure status.

Parameters
pseqSequence object. If NULL, a new sequence object will be created.
chI/O channel.
src1_iovsFirst source I/O vector array.
src1_iovcntSize of the src1_iovs array.
src1_domainMemory domain to which the first source buffers belong.
src1_domain_ctxFirst source buffer domain context.
src2_iovsSecond source I/O vector array.
src2_iovcntSize of the src2_iovs array.
src2_domainMemory domain to which the second source buffers belong.
src2_domain_ctxSecond source buffer domain context.
cb_fnCallback to be executed once this operation is completed.
cb_argArgument to be passed to cb_fn.
Returns
0 if operation was successfully added to the sequence, negative errno otherwise.
